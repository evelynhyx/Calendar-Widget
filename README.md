~ My Calendar Widget ~

A cute minimalistic desktop application that displays the current date!
Includes:
- day
- month
- custom colour themes (blue, pink, beige)

------
I made this project because it's perfect for beginners and I also want to learn how to code desktop applications rather than just websites.
Also, none of the existing widget apps matched my style, so I decided to create my own!

------
I used Electron to create this desktop application. It was honestly more complicated than I thought, especially when working with such a small display screen, which is why I scratched out my original ideas. I created a function that can get the current month and date so that the calendar actually works. I also made it go in front of all other windows so that you can see the date while doing work! I used Canva to make the plaid background.

------
I struggled with javascript because it's pretty new to me and I learned more about it. I also learned about functions and how they can be used to alter things on the desktop app. Adding a to-do list would be too much for this small app (although I found out a few hours after working on it 😭), so I decided to make it minimalistic and cute instead!
