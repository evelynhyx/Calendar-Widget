~ My Calendar Widget ~

A cute minimalistic desktop application that displays the current date!
Includes:
- day
- month
- custom colour themes (blue, pink, beige)

------
I made this project because it's perfect for beginners and I also want to learn how to code desktop applications rather than just websites.
Also, none of the existing widget apps matched my style, so I decided to create my own!

------
I used Electron to create this desktop application.
